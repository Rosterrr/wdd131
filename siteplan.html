Project Site Plan

Site Name

Example Name: Bay Area eBiking Club

This name represents a club that is focused on electric biking in the San Francisco Bay Area.

Optional domain availability: bay-ebike.org

Site Purpose

The site provides a hub for regional e-biking by listing trail and e-bike recommendation information along with a membership information form.

Scenarios

  • What is the best e-bike to get for mountainous areas?
  • Where can I find contact information for the club's directors?

Color Scheme

For the color scheme of the website, I have selected two colors:

  • Primary Color: Green (#4CAF50) - This color will be used for headings (e.g., h1, h2) to provide a fresh and vibrant look.
  • Secondary Color: Orange (#FF9800) - This color will be used for accents, such as links, buttons, and highlights to create visual interest.

Typography

The typography for the website will use the following Google Fonts:

  • Roboto - This font will be used for headings (h1, h2) as it has a clean, modern look with good legibility. It will be used to create a clear visual hierarchy.
  • Open Sans - This font will be used for body text and paragraphs, offering good readability for large chunks of text. It's a versatile font that complements Roboto well.
